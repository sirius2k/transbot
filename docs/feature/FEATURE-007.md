# FEATURE-007: 사이드바 UI (설정 섹션)

## 개요

- **기능명**: 사이드바 UI (설정 섹션)
- **상태**: ✅ 완료
- **분류**: 프론트엔드

## 기능 설명

Streamlit 사이드바를 활용하여 애플리케이션 설정을 별도 영역에 배치. 메인 콘텐츠 영역과 설정 영역을 분리하여 사용성 향상

## 작업(Task) 분해

### Task 7.1: 프론트엔드 - 사이드바 기본 구조

- **분류**: 프론트엔드
- **의존성**: 없음
- **작업 내용**:
  - `st.sidebar` 컨테이너 생성
  - 사이드바 제목/헤더 설정
  - 기본 레이아웃 구성
- **예상 파일**: `app.py`

### Task 7.2: 프론트엔드 - API 키 입력 섹션

- **분류**: 프론트엔드
- **의존성**: Task 7.1
- **작업 내용**:
  - OpenAI API 키 입력 필드
  - 비밀번호 타입 입력 (마스킹)
  - 환경변수 우선 사용 안내
- **예상 파일**: `app.py`

### Task 7.3: 프론트엔드 - 모델 선택 섹션

- **분류**: 프론트엔드
- **의존성**: Task 7.1, FEATURE-003
- **작업 내용**:
  - AI 모델 선택 드롭다운 배치
  - 모델별 설명 추가
  - 기본값 설정
- **예상 파일**: `app.py`

### Task 7.4: 프론트엔드 - 번역 설정 섹션

- **분류**: 프론트엔드
- **의존성**: Task 7.1
- **작업 내용**:
  - 번역 방향 선택 (자동/영→한/한→영)
  - Markdown 지원 토글
  - 기타 옵션
- **예상 파일**: `app.py`

### Task 7.5: 프론트엔드 - 사이드바 스타일링

- **분류**: 프론트엔드
- **의존성**: Task 7.1 ~ Task 7.4
- **작업 내용**:
  - 섹션 구분선 (st.divider)
  - 아이콘 활용
  - 일관된 레이아웃
- **예상 파일**: `app.py`

### Task 7.6: 프론트엔드 - 정보/도움말 섹션

- **분류**: 프론트엔드
- **의존성**: Task 7.1
- **작업 내용**:
  - 앱 버전 정보
  - 사용 가이드 링크
  - 문의/피드백 안내
- **예상 파일**: `app.py`

## 작업 흐름도

```text
Task 7.1 (기본 구조) ──┬──→ Task 7.2 (API 키 입력)
                       │
                       ├──→ Task 7.3 (모델 선택)
                       │
                       ├──→ Task 7.4 (번역 설정)
                       │
                       ├──→ Task 7.6 (정보/도움말)
                       │
                       └──→ Task 7.5 (스타일링) ← 모든 섹션 완료 후
```
