# FEATURE-003: 다양한 AI 모델 선택

## 개요

- **기능명**: 다양한 AI 모델 선택 (GPT-4o, GPT-4o Mini, GPT-4 Turbo, GPT-4, GPT-3.5 Turbo)
- **상태**: ✅ 완료
- **분류**: 백엔드 + 프론트엔드

## 기능 설명

사용자가 번역에 사용할 OpenAI GPT 모델을 선택할 수 있는 기능. 비용, 속도, 품질에 따라 적절한 모델 선택 가능

## 작업(Task) 분해

### Task 3.1: 백엔드 - 모델 정의 및 매핑

- **분류**: 백엔드
- **의존성**: 없음
- **작업 내용**:
  - 지원 모델 목록 정의 (딕셔너리)
  - 표시명 ↔ API 모델명 매핑
  - 기본 모델 설정 (GPT-4o Mini 추천)
- **예상 파일**: `app.py`

```python
model_options = {
    "GPT-4o Mini (추천 - 가성비)": "gpt-4o-mini",
    "GPT-4o (최고 품질)": "gpt-4o",
    "GPT-4 Turbo": "gpt-4-turbo",
    "GPT-4": "gpt-4",
    "GPT-3.5 Turbo (빠름)": "gpt-3.5-turbo"
}
```

### Task 3.2: 프론트엔드 - 모델 선택 드롭다운 UI

- **분류**: 프론트엔드
- **의존성**: Task 3.1
- **작업 내용**:
  - st.selectbox로 모델 선택 UI 구현
  - 사이드바에 배치
  - 기본값으로 추천 모델 선택
- **예상 파일**: `app.py` (사이드바 섹션)

### Task 3.3: 백엔드 - 선택된 모델로 API 호출

- **분류**: 백엔드
- **의존성**: Task 3.1, Task 3.2, FEATURE-001
- **작업 내용**:
  - translate() 함수에 model 파라미터 추가
  - 선택된 모델명으로 API 호출
  - 모델별 에러 핸들링 (모델 가용성 체크)
- **예상 파일**: `app.py`

### Task 3.4: 프론트엔드 - 모델 정보 표시

- **분류**: 프론트엔드
- **의존성**: Task 3.2
- **작업 내용**:
  - 각 모델별 특징 안내 (선택 시 설명 표시)
  - 현재 선택된 모델 강조 표시
- **예상 파일**: `app.py`

## 작업 흐름도

```text
Task 3.1 (모델 정의) ──→ Task 3.2 (선택 UI) ──→ Task 3.3 (API 호출 연동)
                                │
                                └──→ Task 3.4 (모델 정보 표시)
```

## 완료 기준

- [ ] 5개 모델 중 하나를 선택할 수 있음
- [ ] 기본값으로 GPT-4o Mini가 선택됨
- [ ] 선택한 모델로 실제 번역이 수행됨
- [ ] 모델별 특징이 사용자에게 안내됨
