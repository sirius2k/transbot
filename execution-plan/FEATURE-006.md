# FEATURE-006: 번역 결과 듀얼 복사 버튼

## 개요

- **기능명**: 번역 결과 듀얼 복사 버튼 (포맷포함/텍스트만)
- **상태**: ✅ 완료
- **분류**: 프론트엔드

## 기능 설명

번역 결과를 클립보드에 복사하는 두 가지 옵션 제공:
1. **포맷 포함 복사**: Markdown 형식 그대로 복사
2. **텍스트만 복사**: 포맷 제거 후 순수 텍스트만 복사

## 작업(Task) 분해

### Task 6.1: 프론트엔드 - 클립보드 복사 기능 구현

- **분류**: 프론트엔드
- **의존성**: FEATURE-001 (번역 결과)
- **작업 내용**:
  - JavaScript 클립보드 API 활용
  - Streamlit 컴포넌트와 연동
  - 복사 성공/실패 피드백
- **예상 파일**: `app.py`

### Task 6.2: 프론트엔드 - 포맷 포함 복사 버튼

- **분류**: 프론트엔드
- **의존성**: Task 6.1
- **작업 내용**:
  - "📋 복사 (포맷 포함)" 버튼 구현
  - Markdown 원본 그대로 클립보드에 복사
  - 복사 완료 시 토스트 메시지
- **예상 파일**: `app.py`

### Task 6.3: 프론트엔드 - 텍스트만 복사 버튼

- **분류**: 프론트엔드
- **의존성**: Task 6.1
- **작업 내용**:
  - "📄 복사 (텍스트만)" 버튼 구현
  - Markdown 구문 제거 로직
  - 순수 텍스트만 클립보드에 복사
- **예상 파일**: `app.py`

### Task 6.4: 백엔드 - Markdown 스트리핑 유틸리티

- **분류**: 백엔드
- **의존성**: 없음
- **작업 내용**:
  - Markdown 구문 제거 함수
  - `#`, `*`, `[]()`  등 제거
  - 코드 블록 처리 정책
- **예상 파일**: `app.py`

### Task 6.5: 프론트엔드 - 버튼 레이아웃 및 UX

- **분류**: 프론트엔드
- **의존성**: Task 6.2, Task 6.3
- **작업 내용**:
  - 두 버튼 나란히 배치 (st.columns)
  - 아이콘과 레이블 조합
  - 호버 효과 및 클릭 피드백
- **예상 파일**: `app.py`
