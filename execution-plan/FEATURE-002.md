# FEATURE-002: 자동 언어 감지

## 개요

- **기능명**: 자동 언어 감지 (한국어/영어 자동 판별)
- **상태**: ✅ 완료
- **분류**: 백엔드 + 프론트엔드

## 기능 설명

사용자가 입력한 텍스트의 언어를 자동으로 감지하여 번역 방향을 자동 설정하는 기능

## 작업(Task) 분해

### Task 2.1: 백엔드 - 언어 감지 알고리즘 구현

- **분류**: 백엔드
- **의존성**: 없음
- **작업 내용**:
  - 한글 문자 비율 계산 로직
  - 영문 문자 비율 계산 로직
  - 언어 판별 임계값 설정
  - `detect_language()` 함수 구현
- **예상 파일**: `app.py`

### Task 2.2: 백엔드 - 언어별 문자 패턴 정의

- **분류**: 백엔드
- **의존성**: 없음
- **작업 내용**:
  - 한글 유니코드 범위 정의 (가-힣, ㄱ-ㅎ, ㅏ-ㅣ)
  - 영문 알파벳 범위 정의 (a-z, A-Z)
  - 특수문자/숫자 처리 정책
- **예상 파일**: `app.py`

### Task 2.3: 프론트엔드 - 자동 감지 모드 UI

- **분류**: 프론트엔드
- **의존성**: Task 2.1
- **작업 내용**:
  - "자동 감지" 옵션 추가
  - 감지된 언어 표시
  - 수동/자동 모드 전환 기능
- **예상 파일**: `app.py` (사이드바 또는 메인 UI)

### Task 2.4: 통합 - 번역 함수와 연동

- **분류**: 백엔드
- **의존성**: Task 2.1, Task 2.2, FEATURE-001
- **작업 내용**:
  - 자동 감지 결과에 따른 번역 방향 설정
  - translate() 함수에 자동 감지 로직 통합
- **예상 파일**: `app.py`

## 작업 흐름도

```text
Task 2.1 (감지 알고리즘) ──┬──→ Task 2.4 (번역 함수 연동)
                           │
Task 2.2 (문자 패턴) ──────┤
                           │
Task 2.3 (자동 감지 UI) ───┘
```

## 완료 기준

- [ ] 한글이 포함된 텍스트는 한국어로 인식
- [ ] 영문만 있는 텍스트는 영어로 인식
- [ ] 혼합 텍스트는 비율에 따라 주 언어 판별
- [ ] 감지된 언어가 UI에 표시됨
