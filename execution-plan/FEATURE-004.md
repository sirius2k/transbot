# FEATURE-004: 글자수/토큰수 실시간 표시

## 개요

- **기능명**: 글자수/토큰수 실시간 표시
- **상태**: ✅ 완료
- **분류**: 백엔드 + 프론트엔드

## 기능 설명

사용자가 입력한 텍스트의 글자수와 예상 토큰수를 실시간으로 표시하여 API 비용 예측 및 입력량 파악을 지원

## 작업(Task) 분해

### Task 4.1: 백엔드 - 글자수 계산 로직

- **분류**: 백엔드
- **의존성**: 없음
- **작업 내용**:
  - 공백 포함/미포함 글자수 계산
  - `count_characters()` 함수 구현
- **예상 파일**: `app.py`

### Task 4.2: 백엔드 - 토큰수 계산 로직

- **분류**: 백엔드
- **의존성**: 없음
- **작업 내용**:
  - tiktoken 라이브러리 활용 (또는 간단 추정)
  - GPT 모델별 토큰 계산
  - `count_tokens()` 함수 구현
- **예상 파일**: `app.py`
- **의존성 패키지**: `tiktoken` (선택사항)

### Task 4.3: 프론트엔드 - 실시간 표시 UI

- **분류**: 프론트엔드
- **의존성**: Task 4.1, Task 4.2
- **작업 내용**:
  - 입력 영역 하단에 카운터 표시
  - 실시간 업데이트 (텍스트 입력 시)
  - 시각적으로 명확한 레이아웃
- **예상 파일**: `app.py`

### Task 4.4: 프론트엔드 - 비용 예측 표시 (선택사항)

- **분류**: 프론트엔드
- **의존성**: Task 4.2, Task 4.3
- **작업 내용**:
  - 토큰당 비용 계산
  - 예상 API 비용 표시
  - 모델별 비용 차이 안내
- **예상 파일**: `app.py`

## 작업 흐름도

```text
Task 4.1 (글자수 계산) ──┬──→ Task 4.3 (실시간 표시 UI) ──→ Task 4.4 (비용 예측)
                         │
Task 4.2 (토큰수 계산) ──┘
```

## 완료 기준

- [ ] 텍스트 입력 시 글자수가 실시간으로 표시됨
- [ ] 토큰수가 함께 표시됨
- [ ] 입력 영역과 카운터가 시각적으로 연결됨
- [ ] 빈 입력 시 0으로 표시됨
